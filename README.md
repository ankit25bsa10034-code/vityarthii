BMI Calculator

It is simple and easy to use  BMI (Body Mass Index) calculator built with Python's Tkinter library. This program helps you quickly calculate your BMI by entering your height in meters and weight in kilograms and calculate BMI.

Features

1)User-friendly interface for entering height and weight in float

2)Calculates BMI using the standard formula

3)Displays BMI result with two decimal points precision

4)Lightweight and easy to run

How to Use it 

Run the Python program.

Enter your height in meters (e.g., 1.75).

Enter your weight in kilograms (e.g., 70).

Click the "Calculate BMI" button.

See the BMI result displayed below the button.



How It Works
The app uses the BMI formula:
height divided by weight in kg square
 
When you click the button, it reads inputs, computes BMI, and updates the display label with the result.

Future Improvements
